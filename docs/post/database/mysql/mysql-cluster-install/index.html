  command:
    mysqld --ndb-nodeid=4

将相关容器启动起来就可以了。启动顺序为管理节点—>数据节点—>MySQL节点

增加节点

增加SQL节点很简单。 第一步 要修改管理节点的mysql-cluster.conf文件 第二步 是要重启数据节点，所有的数据节点都要重启 第三步 启动新的MySQL节点即可。新节点会连接至集群，同步数据至本地
