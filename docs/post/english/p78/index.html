被动语态表示主语是动作的承受者。只有及物动词有被动语态

使用场景

加粗为常用

  1. 不知道具体执行者是谁

    如中文的"家乡将要建一所学校",只知道建学校，但是谁建的，不知道。使用被动语态

    A school will be built in my hometown

  2. 当动作的执行者是泛指或者不言自明的时候

    图书馆的书不能被带走(大家都默认的规则)

    The books in the libraray must not be taken away

  3. 虽然可以说出动作执行者，但强调的是动作的承受者

    如电脑修好了，强调的不是电脑被谁修好，而是强调电脑修好了，可以用了。

    My computer has been repaired

  4. 出于礼貌，委婉而避免提及自己及对方

  5. 用在文章标题、新闻、广告里面，且常省略be

  6. 动作的执行者不是人时，没有生命的事物与抽象概念的时候

构成

be + 及物动词的过去分词.这里be是助动词，没有意思。不是句子中的系动词。不同时态时，助动词be也要跟着改变，但及物动词一直使用过去分词

以及物动词tell举例:

  • 一般现在时态

    is/am/are + told

  • 一般过去时

    was/were + told

  • 一般将来时

    • will(原来的助动词) + be（被动名在的助动词） + told
    • is/am/are going to + be + told

  • 一般现在时

    is/am/are(原来的助动词) + being（被动句的助动词） + told

  • 现在完成时

    has/have(原来助动词) + been（被动名助动词） + told

否定名

在被动句的助动词后面加not

1

The room was cleaned  -> The room was not cleaned.

疑问问

直接将被动句的助动词移至句首

1
2

The room was cleaned -> Was the room cleaned?
Yes, it was/No, it wasn't.

带情态动词的被句

因为情态动词后面的助动词需要跟原型，所以不管何种时态，都是统一结构

主语 + 情态动词 + be + 及特动词的过去分词
